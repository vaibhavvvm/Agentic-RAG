🎯 Three-Tier Intent Router
Tier 1 — Regex
Instant pattern matching for greetings, small talk, and obvious graph patterns. Zero-cost, zero-latency.
Tier 2 — Keyword
Scored keyword hints. Skips LLM call if confidence exceeds threshold (default: 0.4). Detects vector, graph, or hybrid intent.
Tier 3 — LLM Arbiter
LangChain ChatPromptTemplate + PydanticOutputParser. Classifies into: general_chat, vector_retrieval, graph_retrieval, hybrid_retrieval.
